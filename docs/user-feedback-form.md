# AgriBatch Pay Google Form Intake

## Form title

AgriBatch Pay User Feedback — Level 5 onboarding and product iteration

## Questions

1. Name (short answer, required)
2. Email (short answer, required)
3. Wallet address (short answer, required; Stellar public key)
4. Role (multiple choice: Farmer, Buyer, Cooperative, Auditor, Reviewer)
5. Product rating (linear scale 1–5)
6. What worked well? (paragraph)
7. What was confusing? (paragraph)
8. What should we improve next? (paragraph)
9. Would you use this product again? (yes/no)
